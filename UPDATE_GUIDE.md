# OurVoiceVote Website - Quarterly Update Prompt

## Copy and Paste This Prompt Every 1-3 Months:

---

**PROMPT TO USE:**

"I need you to update my OurVoiceVote civic empowerment website with the most current information. Here's what needs to be refreshed:

1. **Election Dates & Deadlines** - Research and update all voter registration deadlines, early voting dates, and election dates for the next 12 months (federal, state, and local elections)

2. **Voting Rights & Laws** - Check for any new voting legislation, court rulings, or changes to voting rights laws that have been passed since our last update

3. **Key Issues Section** - Review and update the 6 key issue areas (Economic Justice, Healthcare Access, Criminal Justice Reform, Education Equity, Climate & Environment, Voting Rights Protection) with current statistics, recent developments, and new policy proposals

4. **Resources & Organizations** - Verify all external links are still active and add any new important civic organizations or tools that have emerged

5. **State-Specific Information for Texas** - Add or update any Texas-specific voting information, deadlines, ID requirements, and local resources

6. **Contact Information** - Verify Election Protection Hotline numbers and other contact resources are still current

7. **Upcoming Elections** - Add information about any special elections, ballot measures, or important local races happening in the next 6 months

Please research current information, update the HTML file, and provide me with the updated code. The file is currently named 'index.html' in my GitHub repository (blacklogic28/ourvoicevote) and deployed on Netlify at ourvoicevote.netlify.app"

## After Claude Provides the Updated Code:

### Step 1: Update GitHub
1. Go to github.com/blacklogic28/ourvoicevote
2. Click on `index.html`
3. Click the pencil icon (✏️) to edit
4. Replace all content with the new code Claude provided
5. Scroll down and click "Commit changes"

### Step 2: Verify Deployment
1. Go to app.netlify.com
2. Click on your "ourvoicevote" project
3. Wait 30-60 seconds for auto-deployment
4. Click "Open production deploy"
5. Verify the updates are live

### Step 3: Test the Site
- [ ] All navigation links work
- [ ] External resource links open correctly
- [ ] Election dates are current

- [ ] Site loads quickly
